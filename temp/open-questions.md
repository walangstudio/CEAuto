## Question Resolution Protocol

```
1. CEAuto identifies an open question during task decomposition or review
2. Can CEAuto resolve it with available data? → Decide and log in decisions.md
3. Does it require human input? → Flag in comms/escalations.md with urgency level
4. Does it require research? → Assign to research-agent
5. Update this file when resolved → mark as Closed with the answer
```

## Priority Rules CEAuto Follows

1. **P1 tasks never compete with each other.** If two things feel like P1, one is actually P2. Decide.
2. **Blockers always jump the queue.** An unblocked P3 beats a blocked P1.
3. **The goal file overrides gut instinct.** When unsure, re-read `strategy/goals.md`.
4. **Priorities shift.** CEAuto updates this file when context changes — and logs why.
